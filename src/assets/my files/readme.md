
# 🌟 A-6 DigiTools Platform

A modern, responsive **Digital Tools Marketplace** built with **React.js**, Tailwind CSS, and DaisyUI.
Users can browse digital products, view details, manage cart items, and checkout with a smooth and interactive interface.

---

## 🚀 Features

### ✨ Core Features

* **Responsive Navbar** with live cart count indicator
* **Interactive Banner** with key highlights
* **Product Cards** displayed in a 3-column responsive layout
* **Cart & Product Toggle** to switch between views
* **Add / Remove Products** from cart
* **Checkout Functionality** with automatic cart clearing
* **Real-time Toast Notifications** for product actions (using React-Toastify)

### 📊 Informational Sections

* **Stats Section** for important metrics
* **Pricing Plans** with structured layout
* **Workflow Steps Section** to guide users
* **Informative Footer** for navigation and branding

### 📱 Fully Responsive

* Mobile, tablet, and desktop support
* Tailwind CSS grid and flex utilities ensure seamless responsiveness

---

## 🛒 Cart Functionality

* Add products to cart dynamically
* Cart icon shows total item count
* Remove individual items or clear all with checkout
* Toast notifications indicate:

  * Product added
  * Product removed
  * Checkout completed

---

## 🔄 Product & Cart Toggle

* **Products View**

  * Displays all available products
  * Default section when visiting the page

* **Cart View**

  * Displays selected items
  * Shows empty message if cart is empty

---

## 🌐 Technology Stack

* **Frontend:** React.js (ES6+)
* **Styling:** Tailwind CSS & DaisyUI
* **Notifications:** React-Toastify
* **Data:** JSON for products

---

## 🌟 Key Highlights

1. Dynamic product rendering from JSON data
2. Real-time cart updates with React state management
3. Instant user feedback with toast notifications
4. Smooth and responsive UI across devices
