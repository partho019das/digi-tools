## 🌐 Technology Stack

* **Frontend:** React.js (ES6+)
* **Styling:** Tailwind CSS & DaisyUI
* **Notifications:** React-Toastify
* **Data:** JSON for products

---

## 🌟 Key Highlights

1. Dynamic product rendering from JSON data
2. Real-time cart updates with React state management
3. Instant user feedback with toast notifications
4. Smooth and responsive UI across devices

---

## 💡 Optional / Bonus Features

* Instant cart count updates
* “Added to cart” visual feedback via toast
* Automatic cart clearing after checkout
